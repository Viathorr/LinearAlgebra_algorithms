dsa_algorithms.array_sorting_proxy module¶

class dsa_algorithms.array_sorting_proxy.ArraySortingProxy(sorting: ArraySorting)[source]¶

Bases: ArraySorting

Proxy class for sorting arrays, ensuring array validity before sorting. This class acts as a proxy to prevent execution of sorting operation by other sorting classes if the array contains elements other than only integers/floats, or only strings.

sort(array: list[int | float | str]) → None[source]¶

Sorts the given number array in ascending order, and string array in alphabetical order, after validating the array content. The original array is modified, and it should contain only integers/floats or only strings.

Parameters:

array (list[int | float | str]) – The array to be sorted.

Raises:

TypeError – If the array contains elements other than integers, floats, or strings.

Returns:

None

A decorator class that adds functionality to measure the execution time of the sorting algorithm. This class wraps an instance of a class that implements the ArraySorting interface and adds functionality to measure the execution time of the sorting algorithm.

class dsa_algorithms.sorting_abstract_factory.Singleton[source]¶

Bases: type

A metaclass implementing the Singleton design pattern. The Singleton metaclass ensures that only one instance of a class is created and provides a way to access that instance globally. This metaclass should be used as a metaclass for classes that should have only one instance throughout the lifetime of the program.
